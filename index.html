Bienvenue dans le Guide des EVs

Ce guide vous aide à optimiser l'entraînement de vos Pokémon en vous indiquant où trouver les Pokémon qui donnent les EVs (Effort Values) souhaités au safari.

Qu'est-ce que les EVs ?

Les EVs sont des points d'effort qui augmentent les statistiques de base de vos Pokémon. Chaque Pokémon vaincu donne des EVs dans une ou plusieurs statistiques spécifiques.

Comment utiliser ce guide ?

  • Recherche rapide : Utilisez le menu déroulant ci-dessous pour trouver tous les Pokémon donnant une stat spécifique
  • Exploration par biome : Parcourez les différents biomes pour voir quels Pokémon s'y trouvent
